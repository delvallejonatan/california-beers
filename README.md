California Beers
============

This web app shows californians beers generated from [this app](https://github.com/delvallejonatan/beer-app)

Use node `>=6.3` and npm `>=3.10.3`

## Install dependencies


* `npm install`
* `bower install`


## Start development

`npm start`

Go to `http://localhost:8080` in your browser.

## Tests

`npm test`
